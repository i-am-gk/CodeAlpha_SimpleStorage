## Simple Storage Smart Contract

This project is part of the CodeAlpha Blockchain Development Internship.

### Task Description
Create a simple Solidity smart contract that:

- Stores an integer value  
- Allows incrementing the value  
- Allows decrementing the value  
- Allows reading the current value  

### Technologies Used
- Solidity  
- Remix IDE  
- Ethereum Virtual Machine  

### Smart Contract Features

1. Public integer variable to store value  
2. Increment function  
3. Decrement function  
4. Getter function to read value  

### How to Run

1. Open Remix IDE  
2. Compile SimpleStorage.sol  
3. Deploy the contract  
4. Use increment and decrement functions  
5. Call getValue to check result
